# 🚀 FLQUIZ – MERN Quiz Portal

## 📌 Overview

FLQUIZ is a full-stack quiz web application built using the MERN stack.
The application allows users to register, log in, and attempt quizzes with a structured frontend-backend architecture.

---

## 🏗️ Architecture

Frontend (React + Tailwind)
⬇
API Calls (Fetch / Axios)
⬇
Backend (Node.js + Express)
⬇
Database (MongoDB)

---

## 🛠️ Tech Stack

### Frontend

* React.js – UI development
* Redux Toolkit – State management
* Tailwind CSS – Styling & UI

### Backend

* Node.js – Runtime environment
* Express.js – API & routing

### Database

* MongoDB – Data storage

### Authentication

* JSON Web Tokens (JWT) – Session handling
* Bcrypt.js – Password encryption

---

## 📁 Project Structure

FLQUIZ/
┣ client/ → Frontend (React)
┣ server/ → Backend (Node + Express)

---

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

git clone https://github.com/AbdulHadiMs/flquiz.git

---

### 2️⃣ Install Dependencies

Frontend:
cd client
npm install

Backend:
cd server
npm install

---

## ▶️ Running the Application

### Start Backend

cd server
nodemon server

### Start Frontend

cd client
npm start

---

## 📦 Important NPM Packages

### Frontend

* react-router-dom → Navigation
* axios → API requests

### Backend

* express → Routing
* mongoose → MongoDB connection
* jsonwebtoken → Authentication
